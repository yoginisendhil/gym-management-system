# gym management system

A complete console-based Gym Management System built with Java and MySQL. This system manages gym operations including members, trainers, workout plans, equipment, memberships, and payments.

## Features

- Member Management: Register, view, and manage gym members
- Trainer Management: Add and view trainers
- Workout Plans: Create and assign workout plans to trainers
- Equipment Management: Add and track gym equipment
- Membership Management: Create and view memberships
- Payment Processing: Record and view payments

## Technologies Used

- Java 8
- MySQL 8.0
- JDBC
- Maven
